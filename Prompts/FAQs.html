PlacementHub - Screen FAQs

Path: /dashboard

S-1
Q1. What is the primary purpose of the dashboard screen?

A: This screen allows a user to view and manage their dashboard data efficiently within the platform.

Q2. How do I search or filter records on this screen?

A: You can use the search bar at the top or the filter dropdowns to narrow down the dashboard list.

Q3. Can I export the data shown here?

A: Yes, if export is supported for dashboard, an "Export CSV" button will be available at the top right of the table.

Q4. What happens if I encounter an error on the dashboard page?

A: Please refresh the page. If the issue persists, contact the system administrator and mention screen ID S-1.

Q5. Is the data on this screen real-time?

A: Yes, the dashboard data is fetched directly from the database and reflects the most recent updates.

Q6. How do I update an entry here?

A: Click on the 'Edit' icon next to the specific dashboard record to modify its details.

Q7. Are deleted dashboard permanently removed?

A: Usually, deletions are permanent unless an archive feature is explicitly stated for dashboard.

Q8. Why can't I see certain dashboard?

A: Your access is scoped to your role (user). You will only see dashboard assigned to your specific tenant or profile.

Q9. Is this screen mobile-friendly?

A: Yes, the dashboard layout is responsive and will adapt to smaller screens like smartphones or tablets.

Q10. How do I navigate back to the main dashboard from here?

A: You can click the "Home" icon in the sidebar or use the breadcrumb navigation at the top of the dashboard page.
